Project Overview

The Problem

Color blindness affects millions of people worldwide, making it difficult to distinguish certain colors in digital environments. This project aims to develop a user-friendly GUI that allows individuals to customize and adjust on-screen colors, improving visibility and accessibility. By providing real-time color filters and contrast enhancements, this tool will help color-blind users navigate their computers more effectively and comfortably.

Why?

Our motivation for this project comes from personal connections with friends and family members who struggle with color vision deficiency. Seeing firsthand how frustrating and limiting it can be to navigate a world designed for full-color vision, we want to contribute to a more inclusive digital environment. For example, while playing games together, we often encountered challenges due to a lack of player customization options for colorblind users.

Our Solution

Similar to a Microsoft accessibility tool, we plan to have an app that changes the display colors of your screen based on the colorblindness you have. This aims to make similar colors easier to distinguish. We’re not offering a one-size-fits-all solution — we aim to develop a flexible system that empowers users to customize their digital experience based on their specific needs and color perception challenges.

Issues with Current Alternatives

  • Inconsistent Accessibility Across Programs: Many tools are app-specific. Users need consistent support across their entire system.
  • Lack of Support for Desktop Computers: Few accessible and free apps offer robust filtering and profile customization for desktop users.

Design Principles Applied

  • Constraints: Limit user input to valid ranges to avoid unreadable screen settings.
  • Discoverability: Users should find settings easily without extensive instructions.
  • Feedback: Real-time visual changes reflect user adjustments.
  • Mappings: Changes like contrast increases immediately clarify text visibility.
  • Conceptual Model: Familiar controls like sliders and toggles make interaction intuitive.
  • Signifiers: Clear labels, icons, and tooltips guide user decisions.
  • Affordances: Buttons and sliders visually indicate how they can be used.
